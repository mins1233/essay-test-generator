이 프로젝트는 논술형 평가 문항 제작 사이트이다.
배포는 github pasges로 한다.
논술형 평가 문항제작의 입력 및 AI의 출력은 순서는 다음과 같다.



### 1. 교사가 입력하는 것

- 성취기준
    - 예) [9과02-04] 종의 개념과 분류 체계를 이해하고, 생물을 계 수준에서 분류할 수 있다.
- 성취기준별 성취수준 (A, B, C, D, E)
    - 예)
        - A : 종의 개념, 생물 분류 체계, 5계의 특징을 이해하고, 생물을 계 수준에서 분류할 수 있다.
        - B : 종의 개념과 생물 분류 체계를 이해하고, 5계를 예를 들어 설명할 수 있다.
        - C : 종의 개념과 생물 분류 체계를 이해하고, 5계를 말할 수 있다.
        - D : 종의 의미와 생물 분류 단계를 말할 수 있다.
        - E : 종의 의미를 말할 수 있다.
- 성취 수준 분류표 (행 : A, B, C, D, E / 열 : 지식 이해, 과정 기능, 가치 태도)



### 2. 교사의 입력 자료를 보고 AI가 답변으로 주는 것

- 평가 항목 (4~5가지 제시)
  - 예) 
    - 우리 주변의 생물 지도를 분석하기 
    - 생물 5계의 특징을 설명하기
    - 주어진 생물의 특징을 알고 이를 5계로 분류하기
    - 분류 기준을 제시하기
    - 종의 개념을 설명하기
- 평가 요소 (3~4가지 제시)
  - 예)
    - 종의 개념을 정의하고 두 생물이 같은 종인지 판단하기
    - 우리 주변 생물을 분류 기준을 제시하여 5계로 분류하기
    - 특정 생물을 5계에 맞게 분류하고 그 이유를 근거를 들어 서술하기



### 3. 교사가 평가 요소 선정

- 2번에서  AI가 준 '평가 요소' 3~4가지 중, 교사가 원하는 문항 개수 만큼 평가 요소를 선택하고 작성함
  - 예) 평가 요소 중 2가지를 선택한 경우, 아래 2가지를 작성함
    - 종의 개념을 정의하고 두 생물이 같은 종인지 판단하기
    - 우리 주변 생물을 분류 기준을 제시하여 5계로 분류하기
- 문항 제작 시 반영되기 원하는 요청사항 적을 수 있게 함 (없으면 안적어도 되게 함)
  - 예) 수내중학교 주변 분당 탄천 지역에서 볼 수 있는 생물을 제시문에 적어줘. 실제 맥락을 담은 상황을 가정해서 문항을 만들어줘.  



### 4. AI의 문항 제작 (아래 내용 모두 AI에게 지침으로 전달함)

- 성취 기준, 성취 기준 별 성취 수준, 성취 수준 분류표, 평가 항목, 추가 요청 사항을 종합하여 논술형 평가 문항을 제작함

- 논술형 평가 문항은 반드시 '실제 맥락'과 '우리 주변에서 접할 수 있는 실제 상황'을 제시문으로 주고, 평가 요소 내용과 갯수에 맞게 문항을 만듬. 제시문과 문항은 유기적으로 연관성이 있게 제작함

  - 예) 평가 요소를 2개 선정하였기에, 제시문을 주고 관련된 문항 2개를 만듬

    - 문항 1(예시) : 제시문에 언급된 탄천의 청둥오리와 왜가리는 겉모습이 다르지만 모두 '새' 라고 불린다. 생물학적 관점에서  종(Species)의 개념을 정의하고, 두 새가 같은 종인지 여부를 판단하시오. 

      - <작성 조건>  
      - 종의 정의에 반드시 ‘생식 능력’, ‘자연 상태’, ‘자손’ 이라는 키워드를 포함할 것. 

    - 문항 2(예시) : 제시문에 등장하는 생물들 중 “대장균, 소나무, 짚신벌레, 잉어” 를 생물 분류 체계인 5계 수준으로 분류하고, 각 계를 구분하는 주요 기준인 세포의 구조와 영양 방식을 파악하시오. 

      - <작성 조건> 
      - 제시된 4가지 생물을 모두 언급하고, 각각 어느 계에 속하는지 명시할 것. 

      - 세포의 구조는 핵의 유무와 생물을 이루고 있는 세포의 수를 기준으로 서술할 것.
      -  영양 방식에 대해 서술할 때 ‘대장균’은 제외하시오.

- AI가 문항을 제작 할 때 발문에 적절한 '반응지시어'를 사용함. 다양한 반응지시어를 최대한 사용할 수 있게 하며, 과학과 반응지시어는 "과학반응지시어.md"를 참고하게 함 



### 5. AI의 채점 기준 제시

- 생성된 제시문과 문항에 맞는 ''채점 기준''과 "채점기준별 배점" 을 AI가 제작함. "채점 기준"은 문항 수에 맞게함. "채점기준별 배점"은 몇단계로 할지 교사가 선택할 수 있게함.
  - 예) 문항은 2개 -> 채점 기준 2개 / 채점 기준 별 배점 -> 4단계 (A, B, C, D)로 선택한 경우 
    - 채점기준 1. 종의 개념 정의 및 같은 종인지 판단하기
    - 채점기준별 배점
      - A : 종을 자연 상태에서 서로 교배할 수 있고, 생식 능력이 있는 자손을 낳을 수 있는 개체들의 무리로 정확하게 정의하였다. 또한 제시된 두 생물이 자연 상태에서 교배하여 생식 능력이 있는 자손을 낳을 수 없으므로 서로 다른 종이라고 정확하게 판단하고, 그 근거를 논리적으로 설명하였다.
      - B : 두 생물을 서로 다른 종이라고 정확하게 판단하였다. 종의 정의에서 ‘자연 상태’, ‘서로 교배 또는 생식 가능’, ‘생식 능력이 있는 자손’ 중 대부분을 포함하였으나, 한 가지 요소가 빠지거나 다소 불분명하게 표현되었다.
      - C : 두 생물이 서로 다른 종이라는 판단은 맞지만, 종을 단순히 ‘서로 교배하여 자손을 낳을 수 있는 생물’ 정도로 설명하여 자연 상태 또는 자손의 생식 능력에 관한 내용이 빠졌다. 또는 종의 개념은 일부 설명했지만 같은 종인지에 대한 판단 근거가 부족하다.
      - D : 종의 의미나 같은 종인지 여부 중 한 가지만 부분적으로 제시하였다. 겉모습, 사는 곳, ‘새’, ‘물고기’와 같은 일반적인 분류만을 근거로 판단하거나, 과학적 근거 없이 같은 종 또는 다른 종이라고만 답하였다.
    - 채점기준 2. (생략함)





### 6. 주의사항

- AI에게 지침을 제공할 때 작성한 '예'는 예시로 참고만 하는 용도이며, 각 논술형 평가에 맞는 성취기준과 성취기준 별 성취수준, 평가 요소 등을 활용하여 다양한 문항, 제시문을 작성하도록 한다. 위 예시와 비슷할 필요도 없다.
